🔧 技术实现
═══════════════════════════════════════════════════════════════════

1. 智能语言检测
   - 首次访问检测浏览器语言
   - 中文环境 → 显示中文
   - 其他环境 → 显示英文

2. 语言持久化
   - 用户选择保存到 localStorage
   - key: 'language'
   - value: 'zh-CN' 或 'en-US'

3. 回退机制
   - 默认回退语言：中文
   - 翻译缺失时使用回退语言

4. 动态切换
   - 使用 useTranslation Hook
   - 切换后立即生效
   - 无需刷新页面

🚀 使用指南
═══════════════════════════════════════════════════════════════════

1️⃣ 安装依赖
   cd client-h5
   pnpm install

2️⃣ 启动项目
   pnpm dev

3️⃣ 测试语言切换
   1. 访问 http://localhost:5173
   2. 点击左上角语言选择器
   3. 选择 English
   4. 观察页面内容变为英文
   5. 刷新页面，语言保持为英文
   6. 切换回中文，重复测试
