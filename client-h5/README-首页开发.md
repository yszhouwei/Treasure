# TreasureX H5客户端 - 首页开发说明

## 🎯 快速预览

### 方式1：使用启动脚本（推荐）
在项目根目录双击运行：
```
start-h5.bat
```

### 方式2：手动启动
```bash
cd client-h5
pnpm install  # 首次运行需要安装依赖
pnpm dev      # 启动开发服务器
```

然后在浏览器访问：`http://localhost:5173`

## 📝 开发规范

### 命名规范
- 组件：PascalCase（如：`Header.tsx`）
- 函数：camelCase（如：`getUserData()`）
- CSS类：kebab-case（如：`banner-section`）
- 常量：UPPER_SNAKE_CASE（如：`MAX_COUNT`）

### 代码规范
- 使用函数式组件 + Hooks
- TypeScript严格模式
- 遵循ESLint规则
- 组件拆分原则：单一职责

### CSS规范
- 使用CSS变量管理主题色
- 使用Flexbox/Grid布局
- 移动优先响应式设计
- 避免使用!important
