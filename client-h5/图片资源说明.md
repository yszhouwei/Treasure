## 🎨 图片设计规范

### 尺寸规范
- **Banner图**: 800 x 400 px（宽屏比例）
- **商品图**: 400 x 400 px（正方形）
- **使用时**: 自适应容器大小

### 格式选择
- **格式**: SVG（矢量图）
- **优势**: 
  - 无损缩放
  - 文件体积小
  - 支持CSS样式控制
  - 可编程动画

### 色彩方案
- **Banner**: 木纹棕色系（#5c4033 - #a0826d）+ 金色点缀（#D4A574）
- **古董怀表**: 深棕黑 + 银灰金属 + 米黄表盘
- **珍稀邮票**: 青蓝背景 + 彩色邮票（红蓝绿橙）
- **复古相机**: 深灰黑 + 银色金属质感
- **机械键盘**: 深紫蓝背景 + RGB彩虹光效

## 💡 使用示例

### 在React组件中使用
```tsx
// 方式1：作为背景图
<div style={{ 
  backgroundImage: 'url(/images/product-watch.svg)',
  backgroundSize: 'cover',
  backgroundPosition: 'center'
}}>

// 方式2：作为img标签
<img src="/images/product-watch.svg" alt="古董怀表" />

// 方式3：动态导入
import productImage from '/images/product-watch.svg';
<img src={productImage} alt="古董怀表" />
```

### 在CSS中使用
```css
.banner-content {
  background-image: url('/images/banner-bg.svg');
  background-size: cover;
  background-position: center;
}
```

## 📝 图片替换指南

如果需要替换为真实商品图片：

### 1. 准备图片
- 推荐尺寸: 800 x 800 px（商品图）
- 推荐格式: WebP（优先）或 JPG/PNG
- 文件命名: 保持原名称或自定义

### 2. 图片优化
```bash
# 使用工具压缩图片
# - TinyPNG (https://tinypng.com/)
# - ImageOptim (Mac)
# - Squoosh (Web)
```

### 3. 替换文件
- 方式1: 覆盖原SVG文件
- 方式2: 添加新文件并更新引用

### 4. 更新代码
```tsx
// 修改 Home.tsx 中的图片路径
backgroundImage: `url(/images/product-watch.jpg)`
```
